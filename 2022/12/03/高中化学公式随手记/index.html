    学习化学其实是一个整体认识的过程，单单积累一些化学公式和技巧是没有用的（准确来说它对于考试之外是没有什么用处的）

    六大强酸：

    \[\ce{HClO_4 > H_2SO_4 > HI > HBr > HCl > HNO_3} \]

    4大强碱：

    \[\ce{KOH_2 Ca\left(OH\right)_2 Ba(OH)_2 NaOH}\]

    生成氢氧化铁胶体

    \[\ce{FeCl3 + 3H2O ->T[△][] Fe\left(OH\right)3 +3HCl }\]

    钠和氧气的反应

    生成氧化钠

    \[\ce{4Na + O_2 = 2Na_2O }\]

    生成过氧化钠

    \[\ce{4Na + 2O_2 ->T[△][] 2Na_2O_2}\]

    钠和水的反应

    \[\ce{2Na + 2H_2O = 2NaOH + H_2\uparrow}\]

    氧化纳

    氧化纳和和水反应

    \[\ce{Na_2O + H_2O = 2NaOH }\]

    氧化纳和盐酸反应

    \[\ce{Na_2O + 2HCl = 2NaCl + H_2O }\]

    氧化纳和二氧化碳反应

    \[\ce{Na_2O + CO2 = Na_2CO_3}\]

    过氧化钠

    过氧化钠和水反应

    \[\ce{2Na2O2 + 2H2O = 4NaOH + O2\uparrow}\]

    过氧化钠和二氧化碳反应

    \[\ce{2Na2O2 + 2CO2 = 2NaCO3 + O2}\]

    碳酸钠

    碳酸钠和盐酸

    \[\ce{Na2CO3 + 2HCl = 2NaCl + H2O +CO2\uparrow}\]

    碳酸氢钠

    碳酸氢钠和盐酸

    \[\ce{NaHCO3 + HCl = NaCl + H2O + CO2\uparrow}\]

    碳酸氢钠和氢氧化钠

    \[\ce{NaHCO3 + NaOH = Na2CO3 + H2O}\]

    氯

    氯气

    实验室制取氯气

    \[\ce{MnO2 + 4HCl(浓) ->T[△][] MnCl_2 + Cl_2\uparrow +2H2O } \]

    氯气与金属

    \[\ce{2Na + Cl2 ->T[△] 2NaCl}\]

    \[\ce{2Fe + 3Cl2 ->T[△] 2FeCl3}\]

    \[\ce{Cu + Cl2 ->T[△] CuCl2}\]

    氯气和氢气(非金属单质代表)

    \[\ce{H2 + Cl2 ->T[点燃] 2HCl}\]

    氯气和水

    \[\ce{Cl2 + H2O = HCl + HClO}\]
